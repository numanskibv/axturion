# Security Policy

AXTURION is designed for governance-oriented environments.

Security is a structural property of the architecture.

---

## Security Principles

- Workflow isolation enforced in services
- Deterministic state changes
- Explicit transition validation
- No hidden automation
- Audit logging of critical mutations

---

## Reporting a Vulnerability

Do not open public issues for security concerns.

Contact:

security@numanski.nl

Include:

- Description
- Reproduction steps
- Affected version
- Impact estimate

---

## Deployment Responsibility

Security depends on:

- Network configuration
- Database access control
- Reverse proxy configuration
- OS hardening
- Authentication integration
